# OTT Streaming Platform Analytics Dashboard (Power BI)

## Overview

This project presents an interactive Power BI dashboard that analyzes content distribution across major OTT streaming platforms including Netflix, Amazon Prime Video, and Disney+ Hotstar. The goal of the project is to explore trends in digital entertainment content and understand how different platforms distribute movies and TV shows across genres, release years, and regions.

The dataset combines information from multiple streaming platforms, allowing comparative analysis of their content libraries. Using Power BI, the data was cleaned, transformed, and modeled to build an interactive dashboard that highlights key trends in the streaming industry.

The dashboard enables users to explore how content varies across platforms and identify patterns in genre popularity, production trends, and platform strategies.

## Dashboard Features

Platform Comparison

* Distribution of titles across Netflix, Amazon Prime Video, and Disney+ Hotstar
* Comparison of content volume between platforms

Content Type Analysis

* Breakdown of Movies vs TV Shows
* Platform-level distribution of different content types

Genre Insights

* Popular genres across platforms
* Comparative genre distribution

Release Trend Analysis

* Number of titles released by year
* Growth of streaming content over time

Interactive Exploration

* Filters for platform, genre, and release year
* Dynamic visuals enabling deeper exploration of content patterns

## Tools and Technologies

* Power BI
* Power Query (Data Cleaning and Transformation)
* Data Modeling
* Data Visualization

## Key Insights

The analysis highlights how streaming platforms differ in their content strategies. Some platforms focus heavily on movies while others have stronger TV show catalogs. The dashboard also reveals trends in genre popularity and how the volume of streaming content has increased significantly in recent years.
